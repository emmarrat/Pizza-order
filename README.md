## Pizza ordering app with client and admin routes

### On the main page user will be on client's route, but if add '/admin' to the url, user will be transferred to the admin route.

### Admin features: 
 - adding new dishes to the menu, editing and deleting
 - getting orders from the client route with full information
 - deleting completed orders

### User features: 
- making order by clicking on the dish on the menu list
- deleting selected dishes from the cart
- sending order

### Dishes and orders are saving on the Firebase Realtime Database.

### Worked with React on TS, Redux-Toolkit, Axios, React-Router-Dom, Bootstrap, Firebase.

## To run this app:
1. Clone this repository
2. Install all necessary dependencies via ```npm i```
3. Start the app via ```npm start```
